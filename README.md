# Power BI Sales Dashboard

## Project Overview
This project presents an interactive Power BI Sales Dashboard designed to analyze sales performance and support data-driven decision-making.

## Dashboard Features
- Executive Dashboard
- Tactical Dashboard
- Operational Dashboard
- Interactive slicers
- KPI tracking
- Revenue analysis
- Customer insights
- Lead conversion analysis

## Tools Used
- Power BI
- DAX
- Power Query
- Microsoft Excel

## Key KPIs
- Total Revenue
- Total Customers
- Lead Conversion Rate
- Customer Retention
- Sales Growth
